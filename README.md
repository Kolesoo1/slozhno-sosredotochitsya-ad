# Сложно сосредоточиться
Проект создан в учебных целях, для овладения техниками адаптивной вёрстки.

## Возможности

*   **Основная фича 1:** Светлая/тёмная тема.
*   **Адаптивный дизайн:** Корректное отображение на мобильных устройствах, планшетах и ПК.

## Технологии

*   **Фронтенд:** HTML5, CSS3, JavaScript
*   **Инструменты и утилиты:** Git, Prettier, Figma
